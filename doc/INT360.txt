Process


Process


The Integration360 Tool (AKA Int360) is an Integration Engineering application that is used to rapidly set the following settings for a biller on a Software Partner basis.
o   Biller Advanced Configuration Settings
o   FTP Configuration Settings
o   Data Pump Settings
o   Invoicing Settings
Only certain software partners who have been used frequently enough and have been used with enough consistency are included in the Int360 tool. The following is a list of the software partners currently supported by the Int360 tool.
Note: 
This list is subject to Change as new Software Partners are added to the tool.
