Summary
The GoldPoint integration is real-time utilizing EBPP and CloudPay Web Services for Consumer Finance. Web services are used for all integration data transfer except the invoice file and daily reconcilitation/payment file (sFTP). GoldPoint is using the IC BIF format for invoice data and the partner fields are unique per biller (see Technical Design for more details).
