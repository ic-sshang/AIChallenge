Problem Statement
Engineering teams often receive system issues and feature requests with insufficient context, making it difficult to assess and address them effectively. This lack of clarity delays investigation and resolution, leading to inefficiencies and frustration.
üí° 
Solution Overview
GenAI Application:
An AI-powered automation tool integrated with Jira that prompts issue reporters with intelligent, context-specific questions to gather missing or unclear information.
Key Workflow / Features:
Triggered via Jira automation
Retrieves issue data using Atlassian APIs
Uses GPT (via AI Foundry) to generate follow-up questions
Sends questions back to the reporter for clarification
Structured intake process for consistent issue reporting
