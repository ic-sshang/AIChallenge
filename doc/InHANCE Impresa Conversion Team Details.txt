Summary
This document outlines the initial coordination and support structure for conversions involving the InHANCE Impresa platform. While technical integration details are not provided in this data dump, the process emphasizes participation in weekly support calls, use of Confluence documentation, and collaboration with the Harris-InHANCE team.
 Process
Definitions
Auto Pay Job
: Downloads ACH flag from InvoiceCloud and sets dummy bank info.
Balance Job
: Sends current balances from InHANCE to InvoiceCloud every 15 minutes.
Balance All Job
: Weekly job that confirms InvoiceCloud balances match InHANCE.
Confluence
: Internal documentation platform used to track integration steps and technical requirements.
Customer Job
: Downloads updates from InvoiceCloud (paperless, email, phone).
IC Clerk
: Dummy user used to post payments directly to accounts if live posting is enabled.
InHANCE Impresa
: A CIS platform provided by Harris used by municipalities and utilities.
Pending Payments
: Payments held in a queue for manual posting in InHANCE.
PDF Linking
: Preferred method for displaying bill images in InvoiceCloud.
Support Call
: Weekly coordination meeting hosted by InHANCE for active projects.
Track 70
: Internal request to create an Invoice Format GUID.
Key Responsibilities
Biller
:
Coordinate with InHANCE and InvoiceCloud teams as needed.
Choose between pending or live payment posting.
Provide sample bill image for template setup.
Ensure Caselle server access for COWS (if applicable).
Provide biller GUID, web service key, invoice type ID, and request invoice format GUID.
Project Manager (InvoiceCloud)
:
Attend weekly InHANCE support calls.
Track project progress and document updates in SmartSheet.
Coordinate with InHANCE technical contacts.
Submit Track 70 for Invoice Format GUID.
Confirm field mapping in CRM.
Coordinate IVR setup (utilitysearch2).
Ensure BIF customization includes email notification dates.
Provider (InHANCE)
:
Host weekly support calls.
Provide technical guidance and integration support.
Quality Assurance
Project Manager
:
Ensure all project updates are reflected in SmartSheets.
Confirm that any integration issues raised in support calls are resolved.
Roles
Biller
: The organization undergoing the InHANCE Impresa platform conversion.
Project Manager (InvoiceCloud)
: Oversees the conversion process and coordinates with InHANCE.
Provider (InHANCE)
: Hosts support calls and provides technical support.
