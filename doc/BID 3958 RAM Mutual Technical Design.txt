User Interaction and Design
Data Flow - Invoice
RAM Mutual uses Invoice Cloud’s BIF format and sends files via UploadInvoiceFile SOAP API. 
Data Flow - Adjustments
RTDR is used for updating balances. 
Data Flow - Payment
Payments are sent by using Data Pump.
Data Flow - Void
Voids are sent by using Data Pump.
Data Flow - ACH Reject
ACH Rejects are sent by using Data Pump.
Data Flow - Change Log
Customer changes such as paperless and autopay are handled with Data Pump.
Data Flow - Bill Presentment
Bill Presentment is handled with PDF Remote Linking. 
Payer Experience
iFrame Invoice Cloud integration is used for seamless payer experience. 
Cloud Pay v2 is used for guest payments. 
SSOConnect is used for logged in experience and account management. 
Multiuser functionality is used for RAM Mutual. 
Agent Experience
AgentConnect iFrame is used for agent experience when an agent is taking a quote payment or one time insurance premium payment.  
Biller Experience
Biller will use the Biller Portal for any reporting needs they may have.
