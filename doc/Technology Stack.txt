Backend Technologies
.NET 8
 (Preferred)
Description
: Modern, cross-platform framework for building scalable and high-performance applications.
Justification
: Provides better performance, security, and support for Linux-based deployments.
.NET 4.8
 (Alternative)
Description
: Legacy framework for building Windows-based applications.
Justification
: Used when there are technological blockers preventing migration to .NET 8.
Front End Technologies
React 18/19
Description
: A JavaScript library for building user interfaces.
Justification
: Provides a component-based architecture, making it easier to build and maintain complex UIs.
Containerization
Docker
Description
: Platform for developing, shipping, and running applications in containers.
Justification
: Ensures consistency across different environments and simplifies deployment.
Orchestration
Kubernetes (AKS)
Description
: Open-source system for automating deployment, scaling, and management of containerized applications.
Justification
: Provides robust orchestration capabilities, scalability, and resilience.
Databases
Azure SQL
Description
: Structured data storage with support for complex queries and transactions.
Justification
: Reliable and widely used for storing structured data.
Messaging
Azure Service Bus
Description
: Message broker for asynchronous communication between microservices.
Justification
: Ensures reliable message delivery and decouples services.
API Gateway and Ingress
Kong API Gateway
Description
: API gateway that provides features like load balancing, authentication, rate limiting, and logging.
Justification
: Enhances API management and security, ensuring reliable and scalable API traffic handling.
NGINX Ingress Controller
Description
: Manages external access to services within the Kubernetes cluster, providing load balancing, SSL termination, and name-based virtual hosting.
Justification
: Efficiently routes traffic to the appropriate services, enhancing the scalability and resilience of the applications.
CI/CD Tools
Azure DevOps
Description
: Integrated set of tools for CI/CD, version control, and project management.
Justification
: Comprehensive solution with strong integration with Azure services.
Monitoring and Logging
New Relic
Description
: Comprehensive monitoring and observability platform.
Justification
: Provides real-time insights into application performance, infrastructure, and user experience.
Open Telemetry
Description
: Open-source observability framework for generating, collecting, and exporting telemetry data (metrics, logs, and traces).
Justification
: Standardizes observability practices and integrates with various backends, including New Relic.
Fluent Bit
Description
: Lightweight and high-performance log processor and forwarder.
Justification
: Efficiently collects, processes, and forwards logs to various backends, enhancing logging capabilities.
Security
Workload Identity
Description
: Integrates Kubernetes with Azure Active Directory to provide secure identity management for workloads.
Justification
: Enhances security by enabling workloads to authenticate with Azure AD without managing secrets.
Azure Key Vault CSI Driver
Description
: Allows Kubernetes applications to securely access secrets, keys, and certificates stored in Azure Key Vault.
Justification
: Provides a secure and managed way to handle sensitive information within Kubernetes.
Contrast Agent
Description
: Application security agent that provides real-time vulnerability detection and protection.
Justification
: Enhances application security by identifying and mitigating vulnerabilities during  runtime.
