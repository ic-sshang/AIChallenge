Kong OAuth Token 
The following URL is a part of the OAuth plugin and becomes available when enabling the plugin on a route.  
A root route will be used with OAuth plugin global credentials enabled.  Global credentials allow for credentials to be used by all OAuth plugins that have global credentials enabled in a workspace. 
New URL (Kong)
New METHOD
https://api.invoicecloud.com/biller/v1/authentication/oauth2/token
POST, OPTIONS
