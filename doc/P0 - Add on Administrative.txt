Overview


This is the project setup and preparation and project kickoff phase.  The assigned IS will prepare for the Project by provisioning it in SmartSheet, reviewing Project requirements and scope, setting Project priority which includes setting an initial Go-live date. The IS assigned to the Project will also update the Add on Project Schedule to meet the scope of the project.
