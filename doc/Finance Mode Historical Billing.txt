EBPP-6017 As an IC user, I want my new PayPal chargeback fees to be utilized when calculating biller fees and generating a biller statement so I know I'm collecting the money
Related Story
Functional Understanding:
FinMod is a console application that runs monthly to audit and generate monthly report for all the fees calculations and summary.
The program is executed monthly for example it executes on 1 December to prepare a summary or report for the prior month that is 1st November to 30th November.
The program basically gets list of all active billers, and it iterates over each biller to process for each, it calculates all the fees and saves the results into database.
After that the program executes a Stored Procedure to summarize all the charges or fees to generate statements for each biller.
The story EBPP-6017 is to include the Charge back fees along with the existing calculation process.
The fees rate and specification is already there, we just need to use it inside the calculation.
