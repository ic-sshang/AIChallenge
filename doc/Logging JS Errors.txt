The JS errors are reported in New Relic under Browser > Application > JS errors
In IC we use the browser API to do the logging.
https://docs.newrelic.com/docs/browser/new-relic-browser/browser-apis/noticeerror/
 
Sample Code
js
                $errorCustomAttributes.category = "ServiceFeeRecalculation";
                $errorCustomAttributes.requestData = maskedRequest;
                $errorCustomAttributes.requestUrl = url;
                $errorCustomAttributes.error = error;

                $err = new Error('RecalculateServiceFee Error', { cause: error });
                newrelic.noticeError($err, $errorCustomAttributes);
New Relic Query
sql
SELECT * FROM JavaScriptError SINCE 30 MINUTES AGO
