Project Updates
This project is currently using .NET Framework 4.5, it is an easy one to bump up to .NET Framework 4.8.1. 
This does not appear to impact the build at all and pushing the framework version up to latest still results in a successful build of the project.
note
Note - If opening the solution in Visual Studio 2022, the IDE will likely prompt for a .NET version update. Responding positively to the IDE’s suggestion of the version update will push the .NET Framework version up to 4.7.1.
The project will need to be bumped to 4.8.x still by opening the 
sqlproject
 properties by right clicking the project and selecting the 
SQLCLR
 tab.
