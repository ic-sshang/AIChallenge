Intro
The Kong Team AAD Groups are intended to enable the 3 standard levels of User access (Admin, Developer, Reader) per Environment to the Azure Resources that represent the Kong Cluster(s) within each specific Environment. Note the same AAD Groups are used to enable access across Deployment Versions within a specific Environment.
Admin: 
ic-iac-kong-team-admins-{environment}
Developer: 
ic-iac-kong-team-developers-{environment}
Reader: 
ic-iac-kong-team-readers-{environment}
