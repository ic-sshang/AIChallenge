Overview:
This page aims to show how to create a build & release pipeline on Azure DevOps.
Creating the Build Pipeline on Azure DevOps
Go to the Pipeline tab and press on the new Pipeline Button
2. click on Azure Repos GIt
3.  Select the Repository that is going to be built
4. Select the configuration that you want to build and test for the project.
5. Align the YAML file to meet your needs for the build
6. Save and Run
Create the Release Pipeline on Azure DevOps
Click on the Release tab and click the “+ New“ button then “+ New Release Pipeline“
Click on Add an artifact 
3. Add your artifact usually build or Azure artifacts
4.  Add your stages (usually Dev and Prod)
5. select a template or start with an empty job. (I’ll start with an Empty Job)
6. Click on the Task tab to modify the stage tasks
7. Modify the task settings depending on where you are deploying these project and to which manner 
8. Name the pipeline and save the Deploy pipeline in the right folder and click create release
