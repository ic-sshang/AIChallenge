Introduction
The .NET 8 Web API application designed to have multiple endpoints related with retrieving DataPump related Data and handling the DataPump management settings.
One of the endpoints is to provide metrics for graphical representations in the Biller Portal. It retrieves DataPump metrics
In its current form, the API exposes a foundational endpoint:
GET /biller/biller-data-pump/metrics/{billerId}
This endpoint returns key performance metrics such as:
Average Response Time
Error Rate Percentage
Timeout Error Rate
These metrics help billers proactively monitor and address performance issues in their APIs.

 Running the API
The API is designed to be containerized and deployed in a Kubernetes environment. It follows standard .NET configuration practices using 
Microsoft.Extensions.Configuration
. Configuration values can be provided in the following order of precedence:
appsettings.json
Environment variables
