Please reference the following details below when creating a new repository in the SRC project on Azure Dev Ops. 
Libraries:
All new libraries will have the Prefix IC(dot)InsertYourName
Web API:
IC{InsertYourName}API
{Domain}{ServiceName}Service
Web UI:
{Domain}{InterfaceName}UI
Console Apps:
{InsertYourName}
{Domain}{JobName}Job
TCP Service:
{Domain}{ServiceName}Service
Web Applications, APIs, etc:
^^^
(set the standard here for the name guideline)
Guidelines on InsertYourName:
This should be related to the business purpose of the app. The repos can be prefixed with a keyword to simulate grouping. 
Example:
Searching for “MyCoolApp” could bring the following up in the repository search.
MyCoolAppDoesThis
MyCoolAppDoesThat
MyCoolAppAlsoDidThis
