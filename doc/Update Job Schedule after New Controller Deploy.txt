 Job Manager Task
Under Job Schedule -> Job configuration task -> Job manager task. Here is where the bulk of the updates necessary for the Job Schedule are located.
Application Packages
Update to the latest package of the OBD Controller.
Command Line
Update the command line to reflect the application package version.
Environment Settings
Update environment variables as you see fit. The IC_ENV_TYPE is set to 2. If you are planning on running multiple tests, set rerun = true. The billeridlist, will select these billers from the list of billers pulled from the database.
