Description

Deployment flow for helm charts. Runs through the environments sbx, dev, uat, prd-pri, and prd-sec (these are hardcoded into the template)
Runs a stage to calculate the chart version (uses the commit ID), then runs 'jobs/helm-deploy.yaml' targeting each environment.
