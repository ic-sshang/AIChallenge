Select the create button from the Jira bread crumb.
 select "Site Reliability Engineering(SR)" from the 'Project name' dropdown options.
Select Issue type.
Select "sre-altimetrik" from the 'Labels' dropdown options.
Select "Observability" from the 'Work Item Category' dropdown options.
Select the checkbox "Yes" form the ‘Must be monthly’ dropdown options.
Select all the required fields and click on “Create” button.
