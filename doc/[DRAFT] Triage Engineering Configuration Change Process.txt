Purpose
Document the process of how and when to submit items to solutions engineering when a biller or software partner (SP) request a configuration change.
Definitions
Configuration change
Any changes that might affect how any integration behaves, which include but are not limited to:
Advanced Settings
Biller Options
Invoice Type Setting
Data Pump Settings
Stratus/FTP Settings
