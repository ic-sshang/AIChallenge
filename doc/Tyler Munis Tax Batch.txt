Summary
The Tyler Munis Tax Batch integration uses Stratus to transfer files to/from IC and the biller. 
The integration with Tyler Munis Tax is unique in that 
each state has a different invoice file format
. Each integration file mapping will be documented in the technical design with the state name it applies to.
The adjustment file and payment file format is the same across all states.
