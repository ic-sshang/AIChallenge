 Objective
Vulnerability was identified by the IC third party security tools. One the vulnerable items is the “restsharp.105.2.3.nupkg“ package.
This document is intended to fix current vulnerability, this may require to upgrade to another package version along with perform some code changes if that is the case.
After running security scan tool the suggested solution to remove vulnerability is to upgrade current package to version 106.15.0.
 Success metrics
Ton confirm vulnerability has been removed it is necessary to run the security tool again and confirm vulnerability is not reported anymore.
Goal
Metric
NOTE1:
RestSharp is a simple REST and HTTP API Client for .NET. So, must of the repositories that use this package are APIs projects.
NOTE2:
In order to test each repository it is required to explore any endpoint that can be reachable for testing. Also, some repos have a HealthCheck endpoint.
NOTE3: For this document purpose the provided metrics will be related to ICCSRConnectAPI repository
 Call to HealthCheck endpoint and confirm response is successful after doing changes.
HealthCheck Endpoint for :
https://devtest6.invoicecloud.com/agentconnectapi/HealthCheck
Response should return “True” or “Success”
Execute attached file named as "ICCSRConnectAPI.postman_collection.json" using production data and confirm the response is successful.
Confirm vulnerability has been removed by using Contrast tool.
After running Contrast tool scanning for this repo the vulnerability should not be displayed anymore.
 Assumptions
Make sure:
Endpoints to test are reachable from the required environment (the recommendation is to do the testing from Invoice Cloud AVDs)
Confirm contrast agent is running fine and the repository to be scanned is reachable from the tool.
