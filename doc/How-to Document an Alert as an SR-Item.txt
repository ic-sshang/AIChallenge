Dev Support is helping engineering teams handle Ops-genie Alerts. Once an alert has been handled and closed, an SRE-Item should be made to document the steps to resolve the issue. This is a “How-To” article detailing the steps to create an SRE Alert item.
 Instructions
On Jira Click the blue Create Button
Change the Project to Altimetrik SRE (SRE)
Change issue type to task
Under Summary Place the Alert Title
Examples highlighted in blue:
Under Description field place the error message placed on the alert or attached log
Examples:
Assign the item to yourself
Add ESRE under labels
Set Priority to Medium.
Assign to yourself
Click Create.
Once the item has been created. Move the item to work in progress. Click reply to customer and type the steps to resolve the issue. Once all the notes have been provided, move the item to closed. The item can now be used for future reference if a similar alert comes up in the future.
