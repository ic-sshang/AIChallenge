Step Two: Create a UX Research Plan
UX Research Plan
This is the stage where you create goals, hypothesis, and expected outcomes. Decide how the research will be conducted and what deliverables you want out of it
Step Three: Set Up Test
Usability Test Plan
Use this document to write a script, create screener questions if needed. Use this document to align all of the research tasks with the research goals, so that you have a focused direction.
GreatQuestion
Set up the test in GreatQuestion
Step Four: Analyze Results
Usability Test Issue Prioritization
Add the UX research tasks to the first page of the document and weight their impact. Then catalogue all of the findings that come up through testing to the issues tab and assign impact weight to each issue item.
FigJam Research Template
Optional activities to analyze results:
Journey Map
Empathy Map
Step Five: Share Results
Create a slide deck that can be shared with leadership that details the quantitative (50% of users said this) and qualitative (GreatQuestion reel of user feedback)
Step Six: Define Potential Solutions
Meet with triad and share prior research and results. Collaborate to create parameters of proposed solutions. 
FigJam Research Template
Optional activities for refining solutions:
MOSCoW
Journey Map
Empathy Map
Impact Effort Matrix
Designer collaborates with team to create updated workflows and prototypes based on triad takeaways.
Step Seven: Test Potential Solution
Repeat steps 2-5
