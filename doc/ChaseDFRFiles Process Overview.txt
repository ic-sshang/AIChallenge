The ChaseDFRFiles process connects to Chase via SFTP to download zip files. These files are then split based on the header record. We import the following types of files:
PDE0018
: ECP Returns
HPDE0020 / RPDE0020
: Chargebacks
HPDE0036 / RPDE0026
: Pinless Debit Chargebacks
RPDE0022
: Returns
RPDE0029
: Retrievals
RACT0019
: NOCs
RACT0002
: Exceptions (Rejects)
HFIN0010, RFIN0010, HFIN0011, RFIN0011
: Residuals
HACT0010, RACT0010
: Deposit Detail
New Files in Chase Integration Guide - Payfac

ANS0084 - MC Location and Payment Faciliator Fee Analysis
BAI2 - Prior Day Reporting for ACH - Nacha Files
