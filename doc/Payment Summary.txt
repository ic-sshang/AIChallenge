The Payment Summary component provides a comprehensive overview of payment details and amounts during the payment process. It handles total payment calculations, convenience fees, and displays relevant payment information based on configuration.
Core Features
Amount Calculation
Total payment amount
Convenience fees based on payment method
Live agent fees when applicable
Prompt pay discounts
Support for partial payments
Display Configuration
Controlled by BillerOption 154 (ShowPaymentSummary)
Toggle visibility of entire summary
Service fee display options
Support for multiple payment modes:
One-time payments
Scheduled payments
FlexPay payments
Special Cases
Digital wallet payments (simplified display)
EMV/card reader transactions
Multi-invoice payments
Donation handling
Key Components
State Management
InvoicePaymentAmountsState tracks final payment amounts
ChargeBatch collection for multi-invoice processing
Separate handling for convenience fees and live agent fees
Payment Method Integration
Credit Card/EMV display
ACH/EFT summary
PayPal brand displays
Digital wallet simplified view

Validation Rules
Total amount verification against invoice balances
Fee calculation accuracy checks
Payment method specific validations
Multi-invoice amount distribution validation
Email Integration
Receipt configuration based on BillerOption 134
Support for multiple email recipients
Separate handling for payer vs biller notifications
Special Features
Support for live agent fee inclusion
FlexPay payment distribution handling
Donation amount segregation
Digital wallet simplified display logic
