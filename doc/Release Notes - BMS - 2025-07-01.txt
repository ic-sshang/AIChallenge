Release Notes 2025-07-01


MC_RPPS - Feature Enhancements and Streamlined Processes




Enhanced Logging and Error Handling:
 Improved clarity in logging and error management within the SFTP class, making it easier for users to understand and troubleshoot issues.


Consistent Event Naming:
 Refactored SFTP logging to utilize constant event names, ensuring standardized reporting for a smoother user experience.


Clearer Code Structure:
 Variable names in the SFTP class have been updated for better readability, allowing users to follow the code logic more intuitively.


Simplified SFTP Instantiation:
 The instantiation process for the RPPS_SFTP class has been clarified, leading to quicker onboarding and integration efforts.


Removal of Deprecated Functionality:
 Old MC File Upload functionality has been removed due to outdated SFTP credentials, helping users avoid confusion and ensuring a more secure workflow. 




These updates aim to deliver a more robust and user-friendly experience with the MC_RPPS system!
