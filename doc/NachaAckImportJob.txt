Description
NachaAckImportJob
A Dotnet Console app to be deployed in AKS that runs on a schedule to import Nacha Acknowledgement file from Chase SFTP
Primary Functionality
The 
NachaAckImportJob
 is responsible for importing and processing ACH Acknowledgement Files received from external treasury or banking partners via SFTP. These acknowledgement files confirm acceptance, rejection, or validation outcomes of previously submitted outbound NACHA funding files generated by the 
NachaFundingJob
.
Key Features
Scheduled as a Cron Job
.NET 8.0
Importing and processing ACH acknowledgement files via SFTP
