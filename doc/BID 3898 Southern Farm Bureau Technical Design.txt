User Interaction and Design
Data Flow - Invoice
BC will send any invoice information to Invoice Cloud via webservices with the default Guidewire Integration. (SOAP API)
If invoices are created on the fly with Cloud Pay v2, these invoices will be removed if it is sitting out with no payment made after 15 mins. 
Data Flow - Adjustments
Adjustments are handled with RTDR with the default Guidewire Integration. (SOAP API)
Data Flow - Payment
Payments are handled with Data Pump with the default Guidewire Integration. (SOAP API)
Payer Experience
Cloud Pay v2 is being used for processing payments while AgentConnect is being used for redirecting the payer to Autopay and Pay by Text pages. 
Agent Experience
Cloud Pay v2 is being used for processing payments while AgentConnect is being used for redirecting the agent to Autopay and Pay by Text pages. 
Biller Experience
Each state will have permissions to make payments through Biller Portal.
