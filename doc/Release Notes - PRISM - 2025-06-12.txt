Release Notes 2025-06-12


Update to IVRTechV1 for Enhanced Payment Processing




Introduced PaymentSourceID
 – Users can now efficiently track payments made through ACH_Sale and CC_Sale, providing better visibility and control over transactions.


Improved Code Readability
 – Enhanced readability in larger conditional statements and added new methods, making it easier for future updates and maintenance.


Enhanced Null Checks
 – Additional checks throughout the code ensure more robust error handling, leading to a smoother user experience.


Streamlined Parameters
 – Updated methods to pass strings instead of whole objects, resulting in faster processing and reduced memory usage.


New AI Widget Enum Added
 – Integration of a new enum for AI Widget enhances flexibility in payment processing options.




Thank you for your continued support, and enjoy the improved features!
