Page 1: Landing Page
As a Biller Admin, I want to see a list of added domains on the landing page, so that I can easily manage and monitor the domains.
Acceptance Criteria
: The landing page displays a details like "Domain", "Added Date", "Last Modified Date", "Edit", and "View DNS Settings".
As a Biller Admin, I want to see the added date and last modified date for each domain, so that I can track changes and updates.
Acceptance Criteria
: Each domain entry includes "Added Date" and "Last Modified Date" columns.
As a Biller Admin, I want to have an edit option for each domain, so that I can modify domain details as needed.
Acceptance Criteria
: Each domain entry has an "Edit" button that opens the domain modification interface.
As a Biller Admin, I want to have a view option for DNS settings, so that I can see and manage DNS configurations.
Acceptance Criteria
: Each domain entry has a "View DNS Settings" button that navigates to the DNS settings page.
As a Biller Admin, I want to have a button to add a new domain, so that I can expand the list of custom email domains.
Acceptance Criteria
: The landing page includes an "Add Domain" button that opens the domain addition popup.

Page 2: Add Domain Popup
As a Biller Admin, I want to add a new domain through a popup, so that I can quickly configure new custom email domains.
Acceptance Criteria
: Clicking "Add Domain" opens a popup with fields for "Domain" and "Invoice Type".
As a Biller Admin, I want to choose the invoice type for the new domain, so that I can categorize the domain appropriately.
Acceptance Criteria
: The popup includes a dropdown or segmented control for selecting "Supported Invoice Type" and "Donation Invoice Type".
As a Biller Admin, I want to save the new domain configuration, so that it is added to the list of domains.
Acceptance Criteria
: The popup includes a "Save" button that adds the domain to the landing page list and closes the popup.

Page 3: DNS Settings Page
As a Biller Admin, I want to view DNS settings for a domain, so that I can verify and manage DNS configurations.
Acceptance Criteria
: Clicking "View DNS Settings" navigates to the DNS settings page for the selected domain.
As a Biller Admin, I want to have text and an icon for copying DNS settings, so that I can easily copy and paste DNS configurations.
Acceptance Criteria
: The DNS settings page includes a copy icon and text next to each DNS setting for easy copying.
As a Biller Admin, I want to verify the details I have entered so that I can ensure the custom email domain setup is correctly configured.
