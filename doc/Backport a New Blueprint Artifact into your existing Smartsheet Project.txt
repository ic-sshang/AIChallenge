Runbook name
How to Backport a New Blueprint Artifact into your existing Smartsheet Project
Description
Process to link add a new blueprint sheet to a new sheet artifact to an existing project in Smartsheet. This is intended for simple sheets and not ones with references and formulas to other artifacts.

Summary
When artifacts are added to a Smartsheet blueprint they are not added to projects that have already been provisioned. Smartsheet is a snapshot-based provisioning tool and will only include those changes in a go-forward manner. This page guides you to a workaround to include a new sheet artifact into an existing blueprint
Copy Artifact to Personal Workspace
In this step, you will 
copy 
the chosen artifact from the Blueprint folder into your personal workspace so we can link it to your project. 
Steps
Description
Role Responsible
1
Create personal/project folder folder
If you haven’t already done so, create a folder in your personal “Sheets” workspace for backported Project Artifacts. Call it “Project Artifacts”. Within, create a sub-folder with the name of the biller you are backporting the sheet artifact for. 
PM/IM
2
Copy Sheet Artifact
In the Blueprints Workspace, find the artifact you wish to back port to your project. Right-click the artifact and “Save as New”. Name it as you wish, then select the recently created sub-folder in your Sheets workspace in the “Save in” field. Click Save.
IM/PM
Link Artifact to Project Workspace
In this step, you will link the newly copied sheet artifact to your project. 
Steps
Description
Role Responsible
1
Link Artifact to Project Workspace
Navigate to your project Dashboard. In the Toolbar, click File → Properties, then click the the last link in the field called “Location”. Navigate to the sheet called “Link & Resources”
PM/IM
2
Open Links & Resources Sheet and Add Link
Navigate to the sheet called “Link & Resources” and open it. From here you may add or replace a link that points to your copied artifact sheet. Right click on the cell and select Hyperlink. 
Select the option to “Link to other Smartsheet, then select the sheet in your personal workspace you wish to link to. 
IM/PM
3
Display on Dashboard
Return to the Links & Resources sheet, enabled “Display on Dashboard” and select the appropriate Category for the artifact.
