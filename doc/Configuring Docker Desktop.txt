Licensing
You need a license to run Docker Desktop! 
Even though it will run without one, you’d be in breach of contract with Docker. That’s not good. If you need to run Docker Desktop, make sure you get a license.
This page will not run step-by-step through installation or configuration; that’s well documented elsewhere.

WSL Install needed as Docker Desktop Prerequisite
Run the following command in PowerShell with Elevated Permissions:
wsl.exe --install -d Ubuntu
wsl --set-version Ubuntu 2
Hyper V Enabled with Containers to run Windows Containers
Run the following command in PowerShell with Elevated Permissions:
Enable-WindowsOptionalFeature -Online -FeatureName $("Microsoft-Hyper-V", "Containers") -All
Add Yourself to Local docker-users Group
net localgroup docker-users AzureAD\MichaelGriehm /add
