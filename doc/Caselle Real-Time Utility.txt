Summary
The Caselle Real-Time Utility integration is EBPP utilizing web services for all data transfer except invoice data (uses SFTP) and bill images (template or PDF extraction). The integration utilizes Caselle Connect APIs for RTDR and Data Pump (payments, voids and paperless change logs).
Data pump for ACH rejects and autopay change logs is not yet available.
