The UpdateFiservBillerLiveDatesCommand class is responsible for updating the cass live dates of Fiserv billers in the Cass system.
It retrieves the list of Fiserv billers from deposit records, checks if each biller is already in the Cass system, and updates the cass live date if necessary.
If a biller does not have an associated terminal, it creates one and sends it to the gateway. 
If a biller has an associated OBD terminal but no invoice type is tied to that terminal, we would automatically tie it to all invoice types available to the biller.
