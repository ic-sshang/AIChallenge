Intro
Global Groups are the Azure AD Groups created that are scoped to have the ability to read and modify Azure Resources across Application Workloads and Shared Service Workload.
Global Cloud Engineers: 
ic-iac-global-cloud-engineers-{environment}
Global Readers: 
ic-iac-global-readers-{environment}
Global AKS Admins: 
ic-iac-global-aks-admins-{environment}
Group for Global AKS Developers: 
ic-iac-global-aks-developers-{environment}
