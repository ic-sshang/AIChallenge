IAT steps - Description
Summary
Implementation Acceptance Testing - IAT is the process all Implementation Managers must complete after Transcard configures the biller/integrator's SMART Hub instance and provisioned API credentials.
Failed test cases can be discussed with a the Outbound Program Manager (Chris Faria), a Solutions Engineer, or a manager.
Failed test cased may require a JIRA.
