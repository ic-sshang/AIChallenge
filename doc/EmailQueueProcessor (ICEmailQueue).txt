ICEmailQueue is an Azure Function App that sends emails. It’s part of our email queue system.
See 
ICEmailQueue (EmailQueueProcessor)
 for details about where this function app is in Azure, its associated storage accounts, etc.
EmailQueueProcessor contains 4 functions, each of which processes (reads data from the database and uses that data to send an email) a different subset of email queues, based on email queue type ID.
Functions
All 4 functions expect a payload matching the format
 
{biller ID}|{email queue type ID}|{email queue ID (pointer to specific email record)}
For example, a message could be 
2002|109|5807
