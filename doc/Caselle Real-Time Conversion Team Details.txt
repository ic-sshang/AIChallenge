Summary
This document outlines the steps required to convert to real-time integration between Caselle and InvoiceCloud. The process includes installing the COWS module, configuring AR Forms, verifying system requirements, and ensuring the appropriate security licenses are in place. The goal is to enable real-time invoice syncing and payment processing through Caselle’s Connect platform.
 Process
Definitions
AR Forms Definition
: A Caselle configuration that defines how invoice files are generated for InvoiceCloud.
COWS (Connect Online Web Service)
: Caselle’s web service module that enables real-time data exchange.
CRW40000
: Security license for the Cash Receipting Web Service Interface.
General Release Version
: The latest stable version of Caselle Connect software.
Open Items
: Outstanding balances that should be included in the invoice file.
Security License
: Authorization required to enable specific Caselle web service features.
Key Responsibilities
Biller
:
Coordinate with Caselle to install COWS and import AR Forms Definition.
Generate and send invoice file and PDF statement to InvoiceCloud.
Ensure all workstations are updated and licenses are in place.
Caselle (Provider)
:
Install and configure COWS.
Import AR Forms Definition and adjust invoice process to send only Open Items.
Provide required parameters for InvoiceCloud setup.
Project Manager (InvoiceCloud)
:
Confirm all technical components are in place.
Coordinate with Caselle and the biller to validate setup.
Support testing and go-live readiness.
Quality Assurance
Project Manager
:
Confirm COWS is installed and running.
Verify that required licenses (CRW40000 and ARW4000) are active.
Ensure invoice file and PDF statement are received and processed correctly.
Roles
Biller
: The client organization implementing real-time integration with Caselle.
Caselle (Provider)
: The CIS vendor responsible for configuring the COWS module and AR Forms.
Project Manager (InvoiceCloud)
: Oversees the integration process and ensures all technical requirements are met.
Procedure
Initiate Integration Setup
Who:
 Project Manager
Request the biller to contact Caselle to install the Connect Online Web Service (COWS).
Request the AR Forms Definition to be imported and configured to send only Open Items.
Collect Required Parameters
Who:
 Caselle
Provide InvoiceCloud with:
Payment codes (tender types)
CIS Service and PayConnect Service endpoints
Notify InvoiceCloud once parameters are ready for configuration.
Generate and Send Test Files
Who:
 Biller
Generate an invoice file and a matching PDF statement for the same billing cycle.
Send both to InvoiceCloud for validation.
Verify System Requirements
Who:
 Project Manager
Confirm all biller workstations are running Caselle Connect version 2021.02.160 or later.
Ensure COWS is installed and updated on the Caselle server.
Confirm that an admin has access to the server to restart COWS if needed.
Validate Security Licenses
Who:
 Project Manager
Verify that the following licenses are active:
CRW40000 (Cash Receipting Web Service Interface)
ARW4000 (Utility Management Web Service Interface)
If missing, instruct the biller to contact their Caselle CRM to request license activation.
Configure Online Payments Workspace
Who:
 Biller
Ensure the Cash Receipting Workspace used for online payments has the “use for online payments” checkbox enabled.
