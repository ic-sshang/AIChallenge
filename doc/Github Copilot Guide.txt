Prerequisites 
Github Account
 - You will need to 
signup
 for a free Github account using your company email.  Once you have successfully signed up notify, 
https://invoicecloud.atlassian.net/wiki/spaces/ED/pages/edit-v2/3532816385#Account-Request-Template
 , the EngageSmart/Invoice Cloud Github organization administrator Martin Henderson via slack or email ( 
mhenderson@invoicecloud.com
 ) to send you a email invite to join the company organization.  After accepting the invite the administrator will grant your account access to Copilot.  You will use this account for your subscription when configuring the Copilot IDE extensions.   
Recommended GitHub handle format is {business_ unit_abbrev}-{first_letter_of_first_name}{last_name}.  Ex “ic-mhenderson”
IDE Installation - 
Copilot is integrated into your favorite IDE as an extension.  You will need a IDE installation on a company asset (laptop, AVD, Terminal Server).
Visual Studio 2022 17.8+ 
VSCode 1.57+
Please check Mimecast for your Github organization invite emails as they typically get held.
If you were a part of the pilot program you do not need to signup.  You will continue to use the same Github account and tooling provisioned during the pilot.

Configure Visual Studio to use your GitHub account for Copilot
Select Git from the main menu
Navigate to the GitHub node in the tree.  Add your GitHub account.  Visual Studio will open a web browser for you to sign in and grant access to your GitHub account.   Note, Edge has problems with ths handshake it is recommended to have Chrome be your default web browser for this actvity.
