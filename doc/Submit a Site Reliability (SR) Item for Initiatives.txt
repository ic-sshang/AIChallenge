Click the create button in JIRA
Make sure that All Fields are displayed (3 dots on the top right of the item → Show Fields → set the drop down menu to All Fields → Apply)
Select “Site Reliability Engineering (SR)” for 
Project 
and “Story” for 
Issue type
.
Fill out the Summary (title of the Jira item) using the following format:
[Location of Tool] - [Short Description of Tool]
For example: CRM Support Tab - Create a page for CS to convert invoice files
Fill out the Description (body of the Jira item) by describing what is being requested.
For example: Currently, CS cannot convert invoice files without also attempting to upload. This item is a request to create a tool that will allow CS to provide a raw file, select the file format for conversion, and convert the file using that file format. A BIF file should be the deliverable after the tool is used. 
Set the Label to “Initiative” and the Work Item Category to “Serviceability”
Click the “Create Button”
