Invoice Data
SSI
Upon request
Biller may have two invoice files - one for standard billing and one for delinquents. File format is the same, only the name of the file differs.
Biller may have different bill image formats for standard vs delinquent bills. By naming the files differently, IC can determine while bill image format to use for display.
Adjustment File
SSI
Upon request
The Adjustments File (aka Customer Balances file) - sent to Invoice Cloud which contains all of the customers' current balances. The adjustment file can be sent as many times as is needed to Invoice Cloud. Customers are encouraged to run the Daily Process after there have been changes to account balances such as after posting penalties to accounts or whenever they need account balances to update.

Bill images are normally template but billers have the option for PDF Extraction.
