About this library
The Azure Identity library provides 
Microsoft Entra ID
 (
formerly Azure Active Directory
) token authentication support across the Azure SDK. It provides a set of 
TokenCredential
 implementations which can be used to construct Azure SDK clients which support Microsoft Entra token authentication.
