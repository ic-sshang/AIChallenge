Summary
The IMT Spectrum integration is a web service-based integration where IMT sends IC a BIF via web services and pulls payments, ACH rejects and changelog (autopay and paperless) data from IC throughout the day (see technical details for specific timing). 
AgentConnect is used for agent payments.
