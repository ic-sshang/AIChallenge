Open file Explorer and navigate to 
%APPDATA%\Roaming\nuget\NuGet.config
Edit the config file with the following lines
  <packageSources>
    <clear />
    <add key="nuget.org" value="https://api.nuget.org/v3/index.json" protocolVersion="3" />
    <add key="ICQE" value="https://pkgs.dev.azure.com/invoicecloud/QA_Src/_packaging/QE-Framework/nuget/v3/index.json" />
    <add key="ICPackages" value="https://pkgs.dev.azure.com/invoicecloud/_packaging/ICPackages/nuget/v3/index.json" />  
  </packageSources>
Copy and paste the following powershell script 
https://github.com/microsoft/artifacts-credprovider/blob/master/helpers/installcredprovider.ps1
  into a powershell terminal and press enter
If this step was successful, the terminal will print the following message: 
"Credential Provider installed successfully"
Return to the project folder and run 
dotnet restore --interactive
Navigate to the URL provided by the dotnet restore command
Enter the code provided by the dotnet restore command
 URL:/spaces/icinfradoc/pages/2672754726/QE+Team+Nuget+Artifact+Authentication
