Release Notes 2025-06-24


Email Sending Enhancements for Custom Domains




Updated CRM Integration: Now uses MailgunAPIPrivateKey for all new domains, ensuring secure and efficient email sending. 


Enhanced ToolEmailLogs: Improved error tracking and reporting, enabling faster troubleshooting and increasing operational efficiency.


Optimized Domain Retrieval: Streamlined domain fetching directly instead of by email address, leading to quicker setup and reduced delays in sending emails.




Enjoy these improvements designed to enhance your email experience with custom domains!


ICCRM - Biller Email Tool Timeout Exception Resolution




Improved Domain Retrieval: We’ve streamlined the process by fetching domains directly instead of relying on email addresses. This results in faster performance and reduced delay, enhancing your overall efficiency.


Enhanced ToolEmailLogs: Updates to the ToolEmailLogs enhance functionality and clarity in logging email activities, making it easier for users to track communications.


Exception Handling Improvements: Resolved a timeout exception in the Console, ensuring smoother operations without interruptions. Enjoy a more seamless experience while using the tool!




Update BillerPortal to use Configured Alias and DisplayName while sending emails




Custom Domain Email Alias
: Users can now configure email aliases for custom domains, providing more personalized communication.


Improved Email Display Name
: The display name for emails can now be customized, enhancing brand visibility and recognition.


Resolved Comments
: Various comments have been addressed and resolved, leading to a smoother user experience. 




Enjoy these enhancements to your email communication within BillerPortal!


New Functionality to Customize Display Names and Aliases




Enhanced Notification System
: Introduced UI and functionality to send notifications through custom domains, giving users more control and personalization.


Improved Domain Management
: Added a label to clearly indicate the currently used domain, alias, and display name, making it easier to manage settings at a glance.


Parameterized Database Calls
: All database calls in EmailDomainHelper are now parameterized with a wrapper class, enhancing security and preventing SQL injection risks.


User Experience Improvements
: Conditionally disabling the update button prevents unnecessary actions, streamlining the user experience.


Codebase Optimization
: Cleaned up minor issues, removed unused methods, and added comments for better maintainability, resulting in a more robust application.




We’re excited about these updates and believe they will significantly enhance your user experience!
