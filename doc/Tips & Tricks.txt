Renaming multiple files extension tutorial
Step 1. Move files to a local area in your machine
Step 2. Open CMD & change directory to the location of the targeted files
Step 3. Type “ren *(original file type extension) *(new file type extension)“
in this screenshot, I am changing the file type extension from .DUP to .PDF
Append BID, or any string, to the front of all the file names in a directory
Step 1. Move files to a local area in your machine
Step 2. Open CMD & change directory to the location of the targeted files
Step 3. Type Get-ChildItem | Rename-Item -NewName { "<BID>_" + $_.Name }
Get-ChildItem | Rename-Item -NewName { "BID_" + $_.Name }
