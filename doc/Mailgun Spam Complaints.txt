Application Overview
Application Name: MailgunSpamComplaints
Version: 
0.1.10
Description: Cron Application runs every 12 hours and processes any emails marking Invoice Cloud as spam. 
Primary Functionality: The application imports email addresses that identify Invoice Cloud as spam, records these emails in a “Do Not Send List", removes them from paperless status, and syncs to the data pump if applicable.
Key Features: Cron K8s Application, .Net 8.0, Payer, Biller, Paperless, Data Pump
