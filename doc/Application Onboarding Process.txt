A Repo will be created for the Application
Note, we specifically limit the creation Repos to certain Personas as a Governance measure, so this process is followed
An Item will be backlogged and scheduled to add the Application Service to a Kubernetes Cluster
If a different Azure PaaS Resource is to be used then an Item will be Backlogged for the creation of that those Resources.
