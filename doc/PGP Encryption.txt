Receive Jira request
 for autopay registration data
Log onto
 IC-CSTerm server
Create directory 
for the Biller following this structure E:\Cryptography\<Biller Name-Biller ID> \PGP Keys\ (i.e C:\Cryptography\PGP Keys\Alpena Power Company-2942)
Generate random PW 
to be used as the keypair passphrase (use BitWarden to generate and store)
Go to BitWarden, either through the Okta tile or the browser plug-in
Click Vault and go to the “IC-Cryptography Passphrases” Collection
Click the ‘+’ to create new record
Name: < Biller ID-Biller Name>
Password: Click the double arrow circle to generate a password, confirm it has the minimum set
Length: 20
A-Z: Checked
a-z: Checked
0-9: Checked
!@#$%^&*: Checked
Minimum numbers: 1
Minimum special: 1
Avoid Ambiguous characters: Checked
Use the double arrow circle again to generate random string and when satisfied with password, click select
Suggest clicking a few times before selecting password
Click save so the record is saved in the BitWarden repository
Open Kleopatra
Create keypair
. Name will be <Biller Name-Biller ID> (i.e. Alpena Power Company-2942).  Always set keypairs to expire at a commercially reasonable date from generation.  Enter PW from step d as the passphrase
 
  Under advanced settings:
Export public key
 from keypair generated in step f.  Name public key as <BillernamePublickey.asc> (i.e Alpena Power Company Public Key.asc) Store Public key in biller’s directory created in step c
 
Provide public key 
to IM specialist via email.  Third party will be responsible to encrypt the autopay registration file (CSV) with this public key
Obtain encrypted file/import data
Receive notification
 that the encrypted autopay registration file has been uploaded to the biller’s secure upload page
Pull file down 
to IC-KeyGen server in the E:\Cryptography\PGP Keys\<Biller Name-Biller ID>  
Keep both encrypted and decrypted files on Secure upload
Decrypt file 
using the private key for this biller-Jira ID
Import data
 into autopay registration portal

 decrypted file with Eraser immediately upon a completed import
