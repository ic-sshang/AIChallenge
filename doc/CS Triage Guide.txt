This triage guide has been put in place to help our Client Service Representatives in triaging DS-Tickets. 
Process - 
Review the ticket whether this can be handled by CS
 Check for ticket completeness in terms of triage details
In the notes section please add a comment stating CS has completed triage 
Move the ticket to Pending and never set the priority
Set the Pending Reason to Awaiting Eng. Support
Canceling Tickets - 
Tickets can be canceled at this time as well. Canceled tickets will not be touched by DS.
Note:-  
Never set the priority, please add comments if you think the priority should be higher.
