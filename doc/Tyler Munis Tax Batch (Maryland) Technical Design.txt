Real Estate Taxes 
File Summary/Purpose: The purpose of this file is transfer bill/invoice data to IC from the biller
File Extension: .txt
File Naming Convention:
File Delimiter: Fixed Space
Specification: 
Sample File: 
SFTP File Path: N/A - integration uses Stratus
File Transfer Frequency: Four times a year - Once for annual and first installments, one supplemental file, one half year, and one second installments
SPECIAL FILE HANDLING:
The first four characters of each row (position 1-4) determine if it is an annual or semi annual bill. Depending on the bill type, that will determine the fields that are pulled for the values.
PTR6 
= Semi Annual - First Installment
PTR7
 
= Semi Annual - Second Installment
PTR1 
= Annual
PTR8 = 
Half Year
If a value besides one of these is in the first 4 positions 
the file will be failed
 + please contact SE to update documentation and troubleshoot
Two file mappings will be needed:
First installments, annual installments and half year bills 
Create the second installment after the first installment is due
For first installments and annual installments
: If there is a invoice type that is not a first installment, annual installment, or half year bill the entire file will fail as this is not expected.
For second installments
: 
Bills labeled PTR7 will create second installments
File Upload Ordering
 - Annual bill file to create annual bills and first installments → Supplemental bill File as Conveyance File → Supplemental Bill File to create first installments → Regenerate annual bill file → Annual Bill File to create Second Installments.
