For “Deployment Failed” Alert you will need to check the releases page on azure devops and check the release found on the name of the Alert.
Example title of Alert
Once in Azure devops follow the link and path below:
https://invoicecloud.visualstudio.com/Src/_release?_a=releases&view=mine&definitionId=302
Pipelines > Release > “Pipeline Name“ (Example above: ICRTDRService_Staging_Publish)
Then search for and open the specific release that failed.
Check the stages and history tab
If messages are shown like above. This means it has been rejected. Make sure to put a note on the alert and close it.
