Click the create button in JIRA
Make sure that All Fields are displayed (3 dots on the top right of the item → Show Fields → set the drop down menu to All Fields → Apply)
Select “Site Reliability Engineering (SR)” for 
Project 
and “Story” for 
Issue type
.
Fill out the Summary (title of the Jira item) using the following format:
[Name of Product] - [Short Description of Request]
For example: Datapump - Add Additional Logging for Daffron API Request Calls
Fill out the Description (body of the Jira item) by describing what is being requested.
For example: Currently, we are not logging API request calls for Billers setup to use the Daffron Data Pump integration. This item is a request to log the API request calls to Daffron’s endpoint. 
Set the Label to “ESRE” and the Work Item Category to “SRE”
Click the “Create Button”
