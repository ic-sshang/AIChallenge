Use the normal procedure to acquire a new cert and make a pfx out of it
Create the two certs needed to upload to Cloudflare:
Create "SSL Certificate" from pfx file:
openssl pkcs12 -in <pfx file name> -nocerts -out <outbound file name-sslkey.pem> -nodes
Extract private key from pfx file:
openssl pkcs12 -in <pfx file name> -nokeys -out <destination file-private.pem>
Create the cert on Cloudflare:
Login to Cloudflare and go to the site you want to create the cert on
Click on the Invoicecloud account
Click on the 
invoicecloud.com
 site
Click on the down arrow next to SSL/TLS
Select Edge Certificates
Click on Upload Custom SSL Certificate
Paste the SSL Certificate in the SSL Certificate box
Bundle method should be compatible
Paste the private in into the Private key box
Select legacy type on the Legacy Client Support pull down
