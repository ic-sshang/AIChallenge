The sections listed in the page navigation can be used as a high level ‘checklist’ of the configuration updates that need to be made at the time of conversion. 
Summary: Billers will convert from MIPS/APPS to Spectrum and do not require a data update or data conversion to alter account or invoice numbers. Existing account and invoice numbers will not change. The conversion information below relates to the settings in CRM that will need to be updated. 
The biller should have a maintenance window for all pages/screens on the IC site (and no access through any IMT site) for the duration of these changes. 
Integration Partner ID 
Location: CRM > Profile
Change: Update profile FROM ‘IMT Computer Services’ TO ‘IMT Spectrum’
Advanced Configuration
Location: CRM > Invoicing > Advanced Configuration 
No changes to ‘Invoicing’. These options should continue to be enabled:
BIF Customizations
Location: CRM Integration Summary
A BIF Customization for the InvoiceFormatGUID will need to be configured for the IMT Spectrum Integration. 
NOTE: Check with the biller to see if the AutopayCollectionDate and/or EmailNotification BIF customizations need to be updated.
FTP Management 
Location: CRM > Tools > Search/Manage FTP Credentials 
FTP user needs to be setup (credentials setup through normal CRM process). 
FTP paths will need to be setup for IC to retrieve PDFs. The instructions below walk through setting up settings for PDF files as well as a zip files. Both may not be used but are generally recommended in case the biller deviates at any point. NOTE: The ability to give individuals permission to create these tasks was worked in

PDF Settings
Search the biller you’re converting
Navigate to FTP Management
Select the ‘NEW PDF SETTING’
Configure the Setting as follows:
Input the FTP Username you configured
Add ‘\ToInvoiceCloud\PDFs’ to the path name 
Check the box for backup
Select the ‘Move to Filename Pending’ move action
Check the box for active
Click save
Zip File Settings
Search the biller you’re converting
Navigate to FTP Management
Select the ‘NEW ZIP FILE SETTING’
Configure the Setting as follows:
Input the FTP Username you configured
Add ‘\ToInvoiceCloud\PDFs’ to the path name 
Check the box for backup
Check the box for delete zip files 
Check the box for active
Click save
Data Pump Management
Location: CRM > DataPump Management 
Update the Data Pump Management screen to enable/disable the necessary configurations
NOTE: Terminal selections and payment type selections may vary depending on the billers terminal configuration (ex: paypal terminals, OBD, etc.). Non-Payment DataPump Configurations should always be updated with no selected configurations.
