Rules of Engagement
Hold standard Project Kick-off call with Biller. 
After the kick off call email Thomas Curtis at Keystone with the information listed in the Partner Coding Requirements section below. 
After Kick Off Complete the following: 
Submit Stratus Install Request
Once Status is installed notify Keystone with computer or location and paths listed in Status management in CRM (Preferred is to have on billers server. Status has to be on the same location as the Keystone system
Keystone will turn on Invoice cloud, point paths to Stratus and create Web.txt and Custbal.txt files for testing and connection confirmation.
 Submit Bill Template Modification (if applicable):
They have a standard image setup, but we have found they all are not universal with the data. 
The client may need titles renamed. Example is some cases is changing Hydrant Fee to StormWater or Fire Protection to Trash. 
Only titles are modified, the client will know what amount should have the appropriate title.

Overriding Summary of Service Label Names
Overriding Summary of Service Label Names
Can override the following:
WaterMeter1
HydrantFee
TrashCharge
StormWater
Electric
Sewer
SewerCharge
FireProtection
Use the 
Labels
 property to select the summary of service label you want to override and assign it with the value to override
example (overriding the 
FireProtection
 label to display as "Trash Service"):
{
   "TemplateConfig": {
        "SummaryOfServiceOverrides": {
            "Labels": {
                "FireProtection": "Trash Service"
            }
        }
   }
}

Displaying Trash Charge
Displaying Trash Charge
Can enable the display of the Trash Charge, it is set to false (disabled) by default no JSON configuration required
example: 
{
    "TemplateConfig": {
        "DisplayTrashCharge": true
    }
}
Disable Default Water Charge
Disable Default Water Charge
Can disable the default water charge, it is set to false by default no JSON configuration required (default will display the water meter data)
example:
{
    "TemplateConfig": {
        "DisableDefaultWaterCharge": true
    }
}
Display Empty Charges
Display Empty Charges
Removes all empty charges by default (no JSON configuration required), can enable the display of empty charges by setting to true
example:
{
    "TemplateConfig": {
        "DisplayEmptyCharges": true
    }
}
