Integration Summary
Southern Software Tax integration will use the BIF file for invoices and BAF file for adjustments.  The BIF and BAF file will be transmitted through webservices.  Payment Export will be File Type ID 310 IF_Southern Software-Tax.   
Invoice Cloud virtual site will be in use for the integration.
