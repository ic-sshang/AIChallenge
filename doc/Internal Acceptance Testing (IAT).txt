IAT steps - Description
Summary
Implementation Acceptance Testing - IAT is the process all Implementation Managers must complete after all coding or configurations have been completed by IC and all other vendors included in the project. Other vendors include a billing system (CIS), engagement platform (SSO), print vendor, system integrator, etc., 
Failed test cases can be discussed with an Implementation Technical Consultant, a Coach or a manager. 
Failed test cased may require a JIRA.
