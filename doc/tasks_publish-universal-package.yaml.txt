Description

Publishes a package to a universal artifact feed.
Universal packages are pretty much zip files / tarballs
You just provide a directory to push, and the entire contents of that directory get compressed and published to the artifact feed.
Before publication, this task runs a script to ensure that the artifact name and version and compliant with the universal artifact naming standards.
This standard is enforced by the feed itself; it's not our standard.
