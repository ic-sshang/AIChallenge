Architecture Overview
Biller Portal 
Web Application (Frontend)
Technology
: ASP.NET 4.8 WebForms
Role
: Provides the user interface for search input and result display.
Features
:
Search form with filters (e.g., customer name, invoice number).
Results autocomplete list view.
Biller Search API (Backend)
Technology
: ASP.NET Core (.NET 8) RESTful API
Authentication
: JWT Bearer
Role
: Acts as a middle layer between the frontend and Azure AI Search.
Responsibilities
:
Accepts HTTP requests from the WebForm app.
Validates and transforms query parameters.
Constructs Azure AI Search queries using the SDK.
Returns structured JSON responses to the frontend.
Azure AI Search (Search Engine)
Indexes
: 
customer-index
, 
invoice-index
, 
payment-index
Role
: Performs full-text search, filtering, faceting, and scoring.
Features
:
BM25 similarity scoring.
Synonym maps and analyzers (e.g., 
standard.lucene
).
Faceted navigation and sorting.
Secure access managed identities.
