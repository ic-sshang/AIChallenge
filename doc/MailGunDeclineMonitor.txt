Overview
MailGunDeclineMonitor runs in Skybot. The repo is deployed to both devtest and production versions of Skybot.
Validating the job works
The MailGunDeclineMonitor checks for the “-test” environment variable to run its HealthCheck method. This variable can be passed in when doing a manual run on Skybot for both devtest and production.
