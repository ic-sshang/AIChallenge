Creating the Slack Channel 
On your ticket item go to the "More Fields" area
In the More Fields area under "Chat Channel," click Create Channel
Add a descriptive name "DS#-descriptiveissuename"
Create the channel
Guidelines within Slack 
Post the channel name in Idev-Support so the team can join if needed
Add the appropriate people to the DS ticket slack channel
Provide the following information:
What's the current problem? [Be technical and Provide relevant details]
What steps have been taken to mitigate this issue, if any?
What would you like the people in the channel to help with?
Any Relevant logs/ information should be posted with a description
Periodically post updates to the channel when working on the issue to keep all
stakeholders informed of the current status of the issue at hand
