Overview
The IC.Database library is used as a common data access component for Azure SQL databases.   The current version supports connecting to SQL databases via Managed Identity (access tokens) and resolving database enumerations to Key Vault secret database names and servers.  This library can be found in the IC nuget package feed.
Accepted .Net Framework Nuget Version: 
1.0.8157.18623+
Preferred .Net Framework Nuget Version:
 Latest 1.X.X version, as of this writing 1.0.8971.38012
There are versions in the IC feed that should be avoided.  Do not use anything below the accepted package version.  
The 2.X variants are for .Net Core projects and should be considered deprecated.  There is another library for use in .Net Core projects.
