Overview
CodeScene is a code analysis and visualization tool.  It provides actionable insights to effectively reduce technical debt and deliver clean code by cross referencing contextual factors such as code quality, team dynamics, and delivery output.

Configuration
Currently CodeScene has been configured with Projects that group Code Repositories aligning with scrum team ownership.  This allows for members of those teams to easily review the reporting data for their Code Repositories.
