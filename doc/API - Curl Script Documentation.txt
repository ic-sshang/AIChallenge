Curl scripts for use in linux (WSL)
You can find the api key and secret in BitWarden
Add a DNS record:
curl -X PATCH --insecure 
https://api.godaddy.com/v1/domains/ic-rob.com/records
 \
  -H 'Authorization: sso-key api-key:api-secret' \
  -H 'Content-Type: application/json' \
  --data '[{"type": "CNAME","name": "blnk1","data": "
http://www.ic.com
 ","ttl": 600}]'
Remove a DNS record:
curl -X DELETE --insecure 
https://api.godaddy.com/v1/domains/ic-rob.com/records/CNAME/blnk1
 \
  -H 'Authorization: sso-key api-key:api-secret' \
  -H 'Content-Type: application/json' 
Update a DNS record:
curl -X PUT --insecure 
https://api.godaddy.com/v1/domains/ic-rob.com/records/CNAME/blnk1
 \
  -H 'Authorization: sso-key api-key:api-secret' \
  -H 'Content-Type: application/json' \
  --data '[{"type": "CNAME","name": "blnk1","data": "
cloudflare.ic.com
","ttl": 600}]'
