Backup/Export Current Cloudflare DNS Records
Export the DNS settings in GoDaddy
Open the exported file
For all of the CNAME entries pointing to Cloudflare replace the part to the right of the CNAME (example: 
ic-keith.com.cdn.cloudflare.net
.) with pal name (example: pal)
Delete the lines in the ; NS Record section of the file corresponding to the NS servers
Delete the line for SOA
Convert the domain to Full DNS setup in Cloudflare
Go to DNS > Settings.
Select Convert to Primary DNS (this will not affect how your traffic is proxied).
Import the DNS records into Cloudflare
Any duplicate DNS entries will not be changed in Cloudflare. 
For example if there is a CNAME entry in Cloudflare for www pointing to 
pal.invoicecloud.com
, and there is a record in the GoDaddy file for www (
www.invoicecloud.com
)  pointing to 
www.invoicecloud.com.cdn.cloudflare.net
, then the entry would still be there pointing to 
pal.invoicecloud.com
 after the import.
Verify that all of the records imported properly
