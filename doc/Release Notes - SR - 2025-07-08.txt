Release Notes 2025-07-08


Biller Portal - Allow Biller Whitelisting on Biller Login Page




Enhanced Biller Validation:
 The function previously named ValidateBillerWhitelist has been renamed to IsBillerWhitelisted, providing clearer intent and functionality.


Clarity in Logic:
 The Captcha skip condition has been refactored for better comprehension, making it easier for users to understand the login process.


Improved Logging:
 Enhancements to logging within the IsBillerWhitelisted function provide better tracking and diagnostics for improved performance.


Configuration Upgrade:
 Biller whitelist validation has been added and configured, ensuring only authorized billers can log in, boosting security and user trust. 




These updates enhance your experience with added clarity, improved security, and optimized performance!
