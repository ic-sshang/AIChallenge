One of the common mistakes when creating a new miscellaneous file type is forgetting to populate the new file type ID within 
GetPendingFileImportTypeByFileID
. If the new ID doesn’t get populated within here, it’ll fall by default as a “Customer Conveyance” and can prevent the file from successfully importing.
