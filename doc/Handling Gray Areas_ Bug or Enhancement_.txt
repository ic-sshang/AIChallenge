Whenever it is unclear if a code change request should be considered a Bug or Product Request please follow the steps below:
Report the issue as a Bug as outlined in 
How to Create A Prod. Engineering Bug Item
 .
Add the beginning of the description add the the following statement in 
bold
: 
Bug Status Unclear
During Bug Court collaborate with the teams to determine whether this items should be considered a Bug or Enhancement.
If the issue is a bug, remove the “
Bug Status Unclear
” statement from the description, and post a comment stating that the status of the bug has been agreed in bug court.
If the issue is an enhancement, remove the “
Bug Status Unclear
” statement from the description, and post a comment stating that the status of the bug has been agreed in bug court.
