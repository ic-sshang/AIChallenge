1.  Copy the Clone url of the target code repo.
2. Navigate to the ADO archive project.
3. Select the repository dropdown and pick import.
4. Paste the Clone url from step 1 and select requires authentication.
5. Input your username and PAT and import
6. Find the build and deploy pipelines
7. Rename the pipelines with the prefix [DO NOT USE]
8. Disable the pipelines 
9. Delete the target code repository 
10. Add/update repository to archive documentation 
ADO Archive Retired Code Bases - PRJ
 
11. Work with platform to create a work item to remove any related assets on systems or servers
