How To
Steps
Check if the sky bot job 
[Staging-ChaseDFRImportChain]
 has been kicked off at 10:15 AM CT.
Job contains chain of process. 
Check if ChaseDFRFiles_Import was able to establish FTP connection and downloaded file.
Check if ChaseDFRFiles was able to unzip files and split it to the respective directories.
Check if ChaseRetrievalRequests job was successful and mail was sent to Biller
Check if ChaseChargebacks was able to send summary to the biller and same is available on the billing portal.
Check if ChaseGlobalBin was able to import bin range files to the Database.
Check if ChasePCL3FileImport was able to import bin range files to the Database.
Check if ChaseNocs has got any data added to the reject table. Can be identified using correction code “C”
Check if PinlessDebitBinFileImport Job was successful
Check if stored procedure was able to move data from DB master to shard.
Check if we have got any chase ACH rejects import error mail.
Check all the above jobs are finished by 11 am CT or latest by 12 pm CT
Questions to Answer
Did the job start at 10:15 AM CT?
Did the server failed to connect to FTP server?
Did the zip files are downloaded successfully?
Did the unzip files are split it to the respective directories?
Did ChaseRetrievalRequests job failed?
Did ChaseChargebacks data is available on billing portal and mail has been sent to biller?
Did ChaseGlobalBin job was successful and  imported bin range files correctly?
Did ChasePCL3FileImport job was successful and  imported bin range files correctly?
Did ChaseNocs job imported data to reject table?
Did stored procedure moved data to shard level?
