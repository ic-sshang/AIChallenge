Standard Changes
Account Creation
Log into skybot 
http://10.0.6.21:8008/automate-schedule/user_session/new
Head over to the ‘Admin’ tab
In the top left pane find the ‘create user’ button
Fill out the form with the corresponding Data
Enter a temporary password. Sites like 
passwordsgenerator.net
 are a good tool
Check the ‘Expired Password’ box. This ensures that when the user logs on, they are forced to change passwords to something that they can remember a bit easier. 
Account Password Reset
Log into skybot 
http://10.0.6.21:8008/automate-schedule/user_session/new
Head over to the ‘Admin’ tab
Find the corresponding user
Right click the correct user and click ‘reset password’
Enter a temporary password. 
Check the ‘Expired Password’ box. This ensures that when the user logs on, they are forced to change passwords to something that they can remember a bit easier.
