Release Notes 2025-08-06


CirrusInsurance Logo and KeyVault Enhancements




Streamlined Logo Handling: Removed unnecessary logo assignment logic, resulting in faster loading times and improved user experience.


Updated Session Storage for Logos: Refactored logo URL handling to ensure more reliable logo display across sessions.


Fixed Logo Logic in Navigation Header: Addressed inconsistencies in logo retrieval for navigation headers, enhancing overall interface reliability.


Consistent Logo Filenames: Converted all logo image filenames to lowercase, which helps prevent potential display issues.


Refined Cirrus Utility Methods: Improved the GetImageLogo method, making logo retrieval more efficient and effective.


Enhanced Security with Azure KeyVault: Added IC.Azure.KeyVault reference to bolster security measures and protect sensitive data.


Optimized BillerGUID Retrieval: Refactored the retrieval process to utilize the Vault service, increasing security and efficiency.


Current Hardcoding of Variables: Implemented temporary hardcoding for specific variables for smoother functionality until further improvements are made. 




These enhancements collectively provide a more streamlined, secure, and user-friendly experience on the CirrusInsurance demo portal!
