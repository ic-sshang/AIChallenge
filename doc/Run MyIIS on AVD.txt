This article outlines the steps and helpful hints for running MyIIS on AVD.
Set Startup Page for MyIIS Website to 
BillerLogin.aspx
Ensure Environment Variable 
IC_LOGS_DIR
 is set to 
C:\Data\Logs
Ensure you AVD System Assigned Managed Identity has access to the Legacy DevTest KeyVault 
icappsettingsdevtest
 via AAD Group Membership in 
IC-KeyVault-Test-User
Ensure you AVD System Assigned Managed Identity has access to the Legacy DevTest SQL Server (Biller Shared) 
ic-dbtest-east-server1
 via AAD Group Membership in 
ICDB-Test-Apps
Ensure the following HOSTS file entry is added: 
10.31.0.6   icservices
Update the Registry with the following values
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\v4.0.30319]
"SchUseStrongCrypto"=dword:00000001
"SystemDefaultTlsVersions"=dword:00000001
[HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\.NETFramework\v4.0.30319]
"SystemDefaultTlsVersions" = dword:00000001
"SchUseStrongCrypto" = dword:00000001
