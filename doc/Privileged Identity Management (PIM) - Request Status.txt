Checking Request Status
 
Sign in to 
https://portal.azure.com
  with your Azure Credentials (the same ones you use to sign into the VPN).
Search for Azure AD Privileged Identity Management.
Click on “My requests”.
Click on “Privileged access groups (Preview)” to view active requests. On each request there is a button 
that will allow you to cancel the request.
When canceling the request, you’ll see a new notification stating that you’ve cancelled the request.
