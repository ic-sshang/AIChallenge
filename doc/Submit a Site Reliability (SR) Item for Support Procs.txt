Introduction
This document will explain how to create and format a Site Reliability Item when requesting a new Support Proc or updates to an existing one.
Currently, these requests are only being submitted by internal SRE members or Client Service representatives. These 2 groups will follow different templates to create the item, please refer to the appropriate section in this document to follow the necessary steps to submit a request.
Submitting Support Proc Bug As SRE/CS
Create a new SR Jira - Bug & follow the template below to fill out the fields of the item
Summary (Title of Jira Item):
CRMSupportProc - [short summary of issue]
Description (Body of Jira Item):
Function Name:
 [function name as shown in CRM → Support → Support Workbench]
Fault: 
[description of the issue, can include expected vs actual behavior in this section]
Steps To Reproduce: 
[step-by-step guide on how to recreate the bug]
Visual Proof: 
[image or video of the bug and/or its symptoms]

Submitting New Support Proc Request As SRE
Create a new SR Jira - Story & follow the template below to fill out the fields of the item
Summary (Title of Jira Item):
CRMSupportProc - [short summary of data being exposed]
Description (Body of Jira Item):
ProcAlias:
 Retrieve [short description of data being exposed]
DBTypeID:
 [1 (BillerShard) / 2 (DBMaster) / 3 (IC_PerfLogging)]
ProcName:
 [name of SP to be created]
RequiredParams:
 Yes
ProcDescription:
 [brief description of procedure and/or additional information about its parameters/output]
RowLimit:
 500 (Default), can request higher value if necessary
Required Parameters:
 BillerID, StartDate, [include more parameters here if necessary]
Optional Parameters:
 EndDate, [include more parameters here if necessary]
Suggested SQL:
 [sql query which returns the data this procedure should expose]
NOTE: 
Please use previous SPs as reference for the how these are being created!
Employee FullName does not need to be requested as this is handled in code, as long as the ChangeRequestorDesc
 
& UserFKID as present in the returning table.
