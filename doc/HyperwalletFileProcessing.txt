The HyperwalletFileProcessing is a scheduled job that funds biller's Gateway batches using Hyperwallet to disburse the funds via ACH or Wire deposits.   The job has multiple command line options to handle generating One Batch File NACHA (XML) and One Batch File Wire (XML), reading Acknowledgement files, reading Response files, and reading Error files.
