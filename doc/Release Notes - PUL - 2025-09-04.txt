SonarQube - Fix the Security Hotspots for BillerPortal/MyIIS

: Access to encryption key variables has been restricted, ensuring sensitive information is better protected.


Dynamic Key Implementation
: We've replaced hardcoded encryption keys with dynamic keys retrieved from a secure vault, boosting security and flexibility.

: A thorough refactor of encryption key management across various files simplifies the process and enhances maintainability.


Updated SMS Licensing
: The SMS functionality is now upgraded to utilize dynamic license keys, improving security and reliability in messaging.


Elimination of Outdated Constants
: Removing the LICENSEKEY constant and other hardcoded keys helps prevent potential security risks.




These updates significantly enhance the security framework and operational efficiency, offering users a more secure and reliable platform!
