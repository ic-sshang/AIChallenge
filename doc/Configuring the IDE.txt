In VSCode, click the SonarQube icon in the left toolbar
In the window that opens, hit 
Add SonarQube Server Connection
Add our server’s URL (
https://sonarqube.invoicecloud-beta.com
), the token you generated, and a name (you can call it anything you want, like 
IC-SonarQube
)
Hit 
Save Connection
If you’re the first one to work with a repository, 
configure the binding for it
. This only needs to be done once, after which it can be saved and shared with the team.

In VSCode, open the connected mode panel by clicking the SonarQube icon in the left sidebar
Hit the 
+
 (add new binding)
Select the project that corresponds with the repository you’re working on
Hit 
Share Configuration
 in the resulting pop up
Generate a User Token
Visit 
SonarQube
 and login

User
 > 
My Account
 > 
Security
