Option Description
Details
52
Customer Portal - Display Paperless Settings for Non-Registered Customers
By default, the Customer Portal will only display Paperless settings for Registered customers. Some Billers may want those same Paperless settings displayed for Non-registered customers as well. Turning this option ON will enable this functionality.
SET THIS TO ON
64
RTDR - Override 6 Hour Previous RTDR Check
For Billers with RTDR, we normally would not pull a RTDR Balance if we had a successful balance within the past 6 hours. Enable this to Force a refresh.
SET THIS TO ON
84
Enable Web Service Request and Response Logging
If enabled, all web service request payloads to Invoice Cloud APIs will be logged, along with our API responses. Logs are available to be viewed under Tools > Web Service Logs.
SET THIS TO ON
129
Display latest $0 or negative balances in Open Invoices and CloudPayV2
When enabled, the most recent $0 or negative balances will be displayed in the Open Invoices and CloudPayV2. Note: Invoice Type #15 must be enabled for this feature to work.
SET THIS TO ON
132
Disable 24-hour payment check on RTDR for AutoPay Payments
When enabled this option will override the 24-hour hold period for RTDR on invoices with recent payment activity
SET THIS TO ON
152
Disable 24-hour payment check for RTDR
RTDR will not refresh balances on invoices with a payment within the previous 24 hours. When enabled, this option will bypass that check
SET THIS TO ON
