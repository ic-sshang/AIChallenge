Running ICKioskAPI
Please reference the ICKioskAPI Specification documentation in the Box for the method calls available for this API. 
https://invoicecloud.box.com/s/km38oc3v6m6snct40racxvwf8ylrhajt
 (Most Recent as of 12/20/2023). You will also need Postman to call the API. This document assumes the reader knows how to navigate Visual Studio to clone a repository and how to create pull requests in Azure DevOps.
Clone the ICKioskAPI repository.
Make the necessary changes required.
Make a pull request in Azure DevOps
Look up ICKioskAPI in the Pipeline section of DevOps -> Select Build ICKioskAPI
Select Run Pipeline -> Select your branch.
Go to the build -> Get the published artifact.
Go to Deploy ICKioskAPI
Select Run Pipeline -> Specify your pull request’s branch, the devtest environment, and paste the artifact version you got from Build ICKioskAPI-> Click Run
Once the branch has been deployed to devtest, you can begin testing on the term servers
Note:
 Make sure you are testing the changes on Postman in the 
term servers. 
The application API keys can be found on the test biller’s biller portal.
On Postman, make sure to specify the test environment and set the application key
Use the ICKioskAPI Specification document to identify if any parameters need to specified
