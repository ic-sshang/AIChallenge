Endpoint URL (PROD):
https://secure.calmutual.com/CMICInvoiceCloudTest/CMICInvoiceCloudService.svc
To invoke the methods in Calmutual Webservice, the svc file has to be referenced and an object of the webservice has to be defined as follows:
CMICInvoiceCloud.CMICInvoiceCloudServiceInterfaceClient objInvoiceCloudService = new CMICInvoiceCloud.CMICInvoiceCloudServiceInterfaceClient();
The input can be provided by defining the SSMemberInfo object as follows:
CMICInvoiceCloud.AccountBillingInformation objAccountBillingInformation = new CMICInvoiceCloud.AccountBillingInformation();
Web Methods:
GetAccountCurrentBalance(cmicmemberNumber, cmicbillerid, accountNumber):
This web method accepts Member number and Account number as input parameters and returns the Billing information (AccountNumber, MinimumDueAmount, CurrentBalanceAmount, OperationResult) as object.
Please see below are the input properties and their value:
Property Name		Values
cmicmemberNumber	From the specific site
cmicbillerid		        From the specific site
accountNumber		From the specific site
Sample Usage:
CMICInvoiceCloud.CMICInvoiceCloudServiceInterfaceClient objInvoiceCloudService = new CMICInvoiceCloud.CMICInvoiceCloudServiceInterfaceClient();
CMICInvoiceCloud.AccountBillingInformation objAccountBillingInformation = new CMICInvoiceCloud.AccountBillingInformation();
abi = objInvoiceCloudService.GetAccountCurrentBalance(cmicmemberNumber, cmicbillerid, accountNumber)
Properties of AccountBillingInformation Object:
Properties					Data Type	Description
AccountNumber				Varchar(25)	Customer’s Account Number
MinimumDueAmount			Number(9,2)	Current due amount for the account
CurrentBalanceAmount		         Number(9,2)	Current balance amount for the account
OperationResult Succeeded	         bool	
OperationResult Message		 Varchar(50)	
OperationResult List<ErrorDetail>		
ErrorDetail Code			         Numeric(18,0)
