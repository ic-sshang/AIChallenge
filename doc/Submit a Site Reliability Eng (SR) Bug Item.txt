Introduction
This document will explain how to create and format a Site Reliability Engineering Bug Item when there is an issue with a page/tool owned by SRE.
SRE Bug Template for CS
Create a new Jira bug issue type for the Site Reliability Engineering (SR) Project & follow the template below to fill out the fields of the item
Summary (Title of Jira Item):
[name of affected service] - [short summary of issue]
Description (Body of Jira Item):
Page:
 [page where bug is located, obtained in URL]
Fault: 
[description of the issue]
Expected Result: 
[experience that the user expects]
Actual Result: 
[experience that is actually happening]
Steps To Reproduce: 
[step-by-step guide on how to recreate the bug]
Visual Proof: 
[image or video of the bug and/or its symptoms]
Sample SR Bug Item Template
Additional Fields (Details of Jira Item)
Labels: Initiative
Work Item Category: Bug/Defect
Once template & fields are filled out, click the “Create“ button to submit the issue.
