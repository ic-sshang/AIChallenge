Overview:
Remote authentication is a multipurpose API (SOAP or REST) that encompasses around credentials of invoicecloud logins.
Different Types of Uses
Remote Authentication Token:
URL: 
https://www.invoicecloud.com/remoteauthenticationapi/user/RemoteAuthentication
